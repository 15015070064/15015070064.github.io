Windows下安装

  1. 进入Python官网下载

    官网提供了两种系统位数
  • X86：32位系统
  • X86-64：64位系统
    官网提供了三种版本
  • web-based: 透过网络安装的，就是执行安装后才透过网络下载python
  • executable: 可執行文件的，既把要安装的python全部下载好在本机安装
  • embeddable zipfile: zip 压缩档，就是python打包成zip压缩包
  1. 我选择了3.8.1，下载完打开

勾选上 Add Python 3.X to PATH
这样就不用配置环境变量
如果想安装在C盘则选择 Install Now
否则选择 Customize installation

3. 选择好安装路径, 安装就好了
